CIMAGE API Portal

Student Management & Attendance Service Documentation

Available API Endpoints
Method Endpoint Description
POST /create_student/<id>/... Create Student Record
GET /get_student/<id> Fetch Student by ID
GET /all Fetch All Students
GET /db Download SQLite Database
POST /attendance Mark Attendance
GET /getattendance View Attendance Records
GET /downloadattendance Download Attendance CSV
GET /getBD/<start>/<end> Attendance CSV by Date Range
GET /getBD/<start>/<end>/<course> Attendance CSV by Date & Course
